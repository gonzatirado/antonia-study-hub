Resumen Teoría de Redes

El estudio de las redes de computadoras se centra en la interconexión de nodos para el intercambio de recursos y datos. Fundamentalmente, debemos comprender el flujo de información a través de los diferentes medios físicos y lógicos.

Conceptos Clave de la Capa de Enlace

  • Direccionamiento MAC: Identificador único de 48 bits asignado a tarjetas de red (NIC).
  • Protocolo ARP: Resolución de direcciones IP a direcciones físicas dentro de una red local.
  • Encapsulamiento: Proceso de añadir encabezados (headers) a los datos a medida que descienden por el stack.
Ejemplo de Configuración de Interfaz content_copy
interface GigabitEthernet0/0
  ip address 192.168.1.1 255.255.255.0
  no shutdown
  description LAN_G0/0

Recordar que para el examen parcial se enfatizará en la diferencia entre Switching y Routing, así como los dominios de colisión vs. dominios de broadcast.
